🐘 Mobile Mascot Debug Test

此頁面用於測試 Mobile 吉祥物功能是否正常加載。

📋 測試步驟

  1. 清除瀏覽器緩存:
    • Windows: Ctrl + Shift + R
    • Mac: Cmd + Shift + R
  2. 打開開發者工具: F12
  3. 切換到移動設備模式: Ctrl + Shift + M
  4. 訪問主頁面: ./dist/index.html
  5. 檢查控制台: 查看是否有 [Mascot] 開頭的日誌

🔍 預期結果

帶埋我出發啦！
✅ Desktop (>768px): 應該看到 Desktop 吉祥物（右上角固定問候語）
✅ Mobile (≤768px): 應該看到 Mobile 吉祥物（右下角，帶隨機台詞氣泡在左側）
⚠️ 如果看不到 Mobile 吉祥物，請檢查：
  • 瀏覽器是否已清除緩存
  • 是否在移動設備模擬模式下（寬度 ≤ 768px）
  • 控制台是否有 JavaScript 錯誤
