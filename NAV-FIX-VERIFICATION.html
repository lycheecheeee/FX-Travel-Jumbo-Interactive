✅ 問題 2：交互失效（按唔到）（已修復）

根本原因：
全局 CSS 規則 svg { pointer-events: none; } 導致導航欄中的 SVG 圖標無法點擊
修復方案：
1. 添加例外規則，允許導航欄內的 SVG 接收點擊事件：
nav svg,
.nav-actions svg,
#langToggle svg,
#mobileMenuBtn svg,
.mobile-nav-link svg {
  pointer-events: auto;
}

2. 確保導航欄 z-index: 10000 高於其他元素
3. 確保導航欄及其子元素設置 pointer-events: auto
